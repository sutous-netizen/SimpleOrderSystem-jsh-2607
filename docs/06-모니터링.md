# 모니터링

주문에 따른 시료 재고에 대한 정보를 모니터링하는 기능이다. 담당자가 현재 시스템의 상태를 한눈에 파악할 수 있도록 구성한다.

## 주문량 확인

현재 상태별(`RESERVED` / `CONFIRMED` / `PRODUCING` / `RELEASE`) 목록을 확인한다. `REJECTED`는 유효한 주문이 아니므로 무시한다.

## 재고량 확인

각 시료별 현재 재고 수량을 확인한다. 주문 대비 재고 수량에 따라 상태도 표기한다.

- **여유**: 주문 대비 재고 충분 상태
- **부족**: 주문 대비 재고 수량 부족 상태
- **고갈**: 수량이 0인 상태

## 예시 UI 화면

```
[4] 모니터링  2026-04-16 09:32:15

[1] 주문량 확인   [2] 재고량 확인   [0] 뒤로
선택 > 1

상태별 주문 현황
RESERVED   3건
CONFIRMED  8건
PRODUCING  3건  ← 생산라인 대기
RELEASE    18건

재고 현황
시료명               재고    상태 
실리콘 웨이퍼-8인치   480 ea  여유 
GaN 에피택셜-4인치    220 ea  여유 
SiC 파워기판-6인치     30 ea  부족 
산화막 웨이퍼-SiO2      0 ea  고갈
```

※ 위 화면은 이해를 돕기 위한 예시이며, 화면 구성은 자유롭게 결정한다.
