# 생산 라인

생산 라인에 대한 정보를 Display 한다. 주문량에 대한 부족분을 생산하되, 수율 및 오차를 고려하여 시료를 생산한다.

## 계산 공식

- 실 생산량 = `ceil(부족분 / 수율)`
- 총 생산 시간 = `평균 생산시간 * 실 생산량`
- 생산 완료 시 주문 상태 `PRODUCING` → `CONFIRMED` 변경

## 생산 현황 표기

현재 생산 중인 시료에 대한 정보를 표기한다. 표기할 정보 수준(주문 정보, 현재까지의 생산량 등)은 자율적으로 결정한다.

## 대기 주문 확인

생산 라인의 대기열인 생산 큐를 이용하여, 생산 작업을 대기하고 있는 목록을 출력한다.

- 스케줄링 전략: **FIFO**
