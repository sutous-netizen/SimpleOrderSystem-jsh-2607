# 시료 관리

시료(Sample)는 이 시스템의 가장 기본이 되는 단위다. 각 시료는 고유한 이름과 속성을 가지며, 시스템에 등록된 시료만 주문이 가능하다.

| 메뉴 | 의미 |
|---|---|
| 시료 등록 | 새로운 시료를 시스템에 추가. 속성 값: 시료 ID, 이름, 평균 생산시간, 수율 |
| 시료 조회 | 등록된 모든 시료 목록을 확인. 현재 재고 수량도 함께 표시 |
| 시료 검색 | 이름 등 속성으로 특정 시료를 검색 |

## 수율

수율 = 정상적인 시료 / 총 생산 시료

예) 100개 생산 중 정상적인 물품 90개 = 0.9
